    <h1>Image Gallery</h1>
    <p style="color: #363636">A simple application to store your images on your server</p>
    <h4>Installation</h3>
    <p>Download the project, and with your terminal, navigate to the root folder of the project.<br>
        Then you will need to install nodemon to run the server:</p>
    <span style="font-weight: bold; margin-left: 30px;">npm install nodemon</span>
    <p>Edit the <span style="font-style: italic;">.env</span> and set a path to the UPLOADS variable:</p>
    <span style="font-weight: bold; margin-left: 30px;">UPLOADS=/home/username/pictures</span>
    <p>Now you are ready to run the server:</p>
    <span style="font-weight: bold; margin-left: 30px;">npm start</span>
